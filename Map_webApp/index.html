Pharmacy Map 탄생 배경

Pharmacy Maps 웹 서비스를 개발하기 이전에 개별적으로 React 온라인 강좌를 들으며 공부 하던 중 영화 API를 활용하여 영화 소개 웹 페이지를 구현하는 예제가 있었는데 해당 예제를 만드는 과정이 꽤 흥미로웠고 예제를 다 만들고나서 추후 포토폴리오 또한 한번쯤 API를 활용하여 제작하기로 결심하였으며,
그리고 마침 기회가 되어 카카오 맵 API를 활용한 주변 약국 위치 조회 및 위치 안내 웹 서비스인 Pharmacy Maps 개발 계획을 구상하였고, 약 3개월 이라는 기간 동안 본 웹 서비스를 개발에 몰두하여 제작하였습니다.

Pharmacy Map 제작에 사용된 스킬들

HTML5, CSS3, JavaScript, BootStrap 등으로 각 웹 페이지들의 골격을 만들었으며, 지도 API는 국내 지리 정보가 구글 지도에 비해 비교적 정확하게 반영 되어있는 카카오 맵 API를 활용 하였고, 2021년 3월 22일 부로 Github에 업로드하며 정식으로 배포를 개시하였습니다.

Pharmacy Map 기능소개

본 페이지는 Pharmacy Maps 웹 서비스의 메인 페이지로 해당 웹 서비스의 소개글, 소개글 부분 하단에 두 가지 버튼이 구현 되어 있는데 하단의 "지도 메뉴로 이동" 버튼을 누르시거나 상단 메뉴 바의 "지도" 메뉴를 클릭 하시면 본격적으로 지도 서비스를 이용하여 내 주변 약국 위치들과 각 약국들의 상세정보(약국들의 상호명, 연락처, 주소) 등이 지도 상에 표시되어 확인 하실 수 있으며, Pharmacy Maps 개발자의 개발 일지를 확인 해보고 싶으신 방문자 여러분들은 상단 메뉴바의 "개발 일지"를 클릭 하시거나, 하단의 "개발 일지로 이동" 버튼을 클릭하면 해당 페이지로 이동합니다.
