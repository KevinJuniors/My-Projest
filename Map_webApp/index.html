  • Pharmacy Maps 기능 소개

    Pharmacy Maps는 카카오 맵 API 기반으로 제작되어진 약국 위치 조회 웹 서비스입니다. 기존 지도 웹 서비스와 유사하게 각 지역 별로 위치 해 있는 실제 약국 위치를 지도 상에 표시해주고 약국 위치를 표시하는 마커 이미지를 클릭하면 해당 약국의 상세 정보(주소, 연락처)를 지도 상에 표시 하는 기능을 제공하는 웹 서비스 입니다.

  • Pharmacy Maps 지도 웹 서비스를 이용 하시려면 지도 페이지로 이동 하시면 됩니다

하단의 아이콘을 클릭하시면 해당 지도 웹 사이트로 이동하여 자세한 위치 정보를 검색 하실 수 있습니다
